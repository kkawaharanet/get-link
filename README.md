# Copy Link

Copy the current page's URL and title to your clipboard in Markdown, HTML, and more.

## Install

[Copy Link - Chrome Web Store](https://chromewebstore.google.com/detail/copy-link/jjojpplkebencpghmklalgeiemcapkpe)

## Build

```bash
$ npm i
$ npm run zip
```
